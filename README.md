# Pizza Sales Analysis Project

##  Project Overview
This project focuses on analyzing pizza sales data to extract meaningful business insights using **SQL**.  
The dataset contains information about orders, pizzas, pizza types, and order details.

The goal is to answer real-world business questions such as total revenue, most popular pizzas, order trends, and category-wise performance.

---

##  Dataset Description
The project uses the following datasets:

- **orders.csv** – Contains order date and time details  
- **order_details.csv** – Contains pizza-wise order quantities  
- **pizzas.csv** – Contains pizza size and price information  
- **pizza_types.csv** – Contains pizza names and categories

##  Key Analysis Performed
Some of the insights extracted include:
- Total number of orders placed
- Total revenue generated
- Highest-priced pizza
- Most commonly ordered pizza size
- Top 5 most ordered pizza types
- Category-wise sales distribution
- Hourly order distribution
- Revenue contribution by each pizza type
- Cumulative revenue analysis over time
